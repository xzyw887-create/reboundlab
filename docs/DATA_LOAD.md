# Без лимита: до полной загрузки всех пар

```bash
nohup caffeinate -dimsu bash scripts/backfill-watchdog.sh 0 1095 >> logs/backfill-watchdog.log 2>&1 &
```

- **`0`** — без ограничения по часам (крутится, пока не загрузится всё)
- **`12`** — после 12 ч автоматически **перезапускается в режиме 0** (без остановки)

## Оптимизации (2026-07-18)

- Коммит после каждого чанка (без deadlock)
- Пропуск уже полных пар (`skip complete`)
- Задержка API 0.08 с (было 0.15)
