## Как запустить реальные цены (вариант A)

Нужен Docker. На Mac мы используем **Colima** (уже установлен).

**Одна команда — всё само:**

```bash
cd /Users/wasya8/Projects/reboundlab
bash scripts/start-variant-a.sh
npm run web:dev
```

**Или вручную по шагам:**

```bash
# 1. Docker (если выключен)
colima start --cpu 2 --memory 4 --disk 20
export DOCKER_HOST=unix://$HOME/.colima/default/docker.sock

# 2. База данных
cd /Users/wasya8/Projects/reboundlab
npm run docker:up

# 3. Загрузить цены BTC, ETH, SOL (7 дней)
bash services/market-data/run.sh backfill 3 7

# 4. Веб-интерфейс
npm run web:dev
```

В форме слева выберите **период** (даты) и нажмите **«Запустить бэктест»** — расчёт идёт по реальным минутным свечам Binance Futures.

## Как запустить веб-интерфейс с графиками

```bash
cd /Users/wasya8/Projects/reboundlab
npm install          # один раз
npm run web:dev      # запуск
```

Откройте в браузере: **http://localhost:3000**

1. Настройте параметры слева
2. Нажмите **«Запустить бэктест»**
3. Увидите: график свечей + маркеры сделок, кривую PnL, таблицу
