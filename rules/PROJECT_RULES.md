# PROJECT RULES — ReboundLab

## Запрещено без разрешения

- Менять архитектуру
- Рефакторить чужие модули
- Добавлять npm/pip пакеты
- Создавать новые сервисы
- Пушить в main напрямую
- Хранить секреты в коде

## Обязательно

- Market Data независим от бэктестера
- Бэктестер не обращается к API бирж
- Только реальные исторические данные
- Кросс-маржа: ликвидация = весь банк = 0
- Комиссии и funding в расчётах
- Дисклеймер на всех результатах

## Подписки

- Единый UI, разные feature flags
- Trial: 10 прогонов Starter
- Quota check перед каждым backtest run

## Данные

- Все USDT-пары с историей ≥365 дней
- 1m ticks — основа для входа
- Incremental sync only

## Юридическое

- Не финансовая рекомендация
- Оферта + privacy policy перед оплатой
- ИП/самозанятый — консультация перед launch

## Агенты (Cursor office)

- Только для владельца, не для пользователей
- Каждый агент — своя зона знаний
